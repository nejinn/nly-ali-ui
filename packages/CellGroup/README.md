# CellGroup 单元格组

## prop

|  prop   |  类型  |  默认值  | 描述 |
|  ----  | ----  |----  | ----  |
| title  | string |  | 单元格组标题 |
| border  | bool |  | 边框 |
| className  | string |  | 自定义 css 类 |
| style  | string |  | 自定义式样 |
| titleClass  | string |  | 标题自定义 css |
| titleStyle  | string |  | 标题自定义式样 |

## slot

|  name   | 描述 |
|  ----  | ----  |
| default  | 默认插槽 |
